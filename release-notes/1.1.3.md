# CloakBrowser Desktop 1.1.3

## 更新概览

本版本修复 `v1.1.2` 云端多平台打包中 macOS 安装包构建失败的问题，并继续保留 Windows、macOS、Linux 多系统多 CPU 架构 Release 打包矩阵。

## 打包修复

- 新增 macOS 专用 `build/icon.icns` 图标资源。
- macOS 打包配置从 PNG 图标切换为 `.icns` 图标，适配 electron-builder 的 macOS 应用图标要求。
- Windows 安装协议继续使用 RTF 文件，避免 NSIS 协议页中文乱码。

## Release workflow 改进

- 保留 Windows x64、macOS x64、macOS arm64、Linux x64、Linux arm64 五个构建目标。
- 将 GitHub Artifact Attestation action 升级到 `v4.1.1`，减少 Linux job 中的 Node 20 deprecation warning。
- Release 仍然要求完整版本说明文件，发布页面不再使用自动生成的提交记录。

## 安装包列表

- Windows x64：NSIS 安装包。
- macOS x64：DMG 与 ZIP 安装包。
- macOS arm64：DMG 与 ZIP 安装包。
- Linux x64：AppImage 安装包。
- Linux arm64：AppImage 安装包。

## 安全说明

当前如果尚未配置商业代码签名证书，浏览器、Windows 或 macOS 仍可能提示安全确认。请仅从官方 GitHub Release 页面下载安装包，并根据系统提示自行确认来源。
