# CloakBrowser Desktop 1.1.6

## 更新概览

本版本新增 API 接口文档页面、内核镜像加速下载功能，并改进了 CDP 远程调试地址。

## 新增功能

### API 接口文档页面

- 新增「API 接口」导航菜单，展示所有本地 HTTP 接口的调用方式。
- 包含环境管理、代理管理、健康检查等接口的请求方法、路径、请求体和 curl 示例。
- 示例代码支持一键复制。

### 内核镜像加速下载

- 内核下载页面新增「镜像加速地址」设置卡片。
- 默认使用 `https://v4.gh-proxy.org/` 加速地址，可自行修改。
- 镜像地址保存在本地配置文件中，应用启动时自动加载。
- 下载时自动拼接完整的 GitHub 下载路径，无需手动配置。

## 改进

### CDP 远程调试地址

- 启动环境后复制的 CDP 地址改为 WebSocket 格式：`ws://127.0.0.1:PORT/devtools/browser/xxxxx`
- WebSocket 地址可直接用于自动化脚本连接浏览器，无需额外拼接。
- 「更多」菜单新增「复制 CDP 地址」选项，未启动时提示先启动环境。
- CDP 地址帮助图标显示提示：CDP 调试地址是脚本连接浏览器最直接的方案。

### 国际化

- 新增 API 接口页面的中英文翻译。
- 新增「复制」通用翻译键。

## 修复

- 修复 `--no-sandbox` 命令行警告（CloakBrowser 内部参数，暂无法移除）。

## 保留的发布目标

- Windows x64：NSIS 安装包。
- macOS x64：DMG 与 ZIP 安装包。
- macOS arm64：DMG 与 ZIP 安装包。
- Linux x64：AppImage 安装包。
- Linux arm64：AppImage 安装包。

## 安全说明

当前如果尚未配置对应平台的商业代码签名证书，浏览器、Windows 或 macOS 仍可能提示安全确认。请仅从官方 GitHub Release 页面下载安装包，并根据系统提示自行确认来源。
