# CloakBrowser Desktop 1.1.4

## 更新概览

本版本继续修复 macOS arm64 云端打包稳定性问题。`v1.1.3` 中 Windows 与 Linux 构建已正常产出，但 macOS arm64 job 在 GitHub Actions 的 `macos-latest` runner 上失败。

## 打包修复

- 将 macOS arm64 runner 从 `macos-latest` 固定为 `macos-15`。
- 避免 `macos-latest` 迁移到 macOS 26 期间带来的环境漂移风险。
- 保留 macOS x64 的 `macos-15-intel` runner。
- 保留 macOS 专用 `.icns` 图标资源。

## 发布目标

- Windows x64：NSIS 安装包。
- macOS x64：DMG 与 ZIP 安装包。
- macOS arm64：DMG 与 ZIP 安装包。
- Linux x64：AppImage 安装包。
- Linux arm64：AppImage 安装包。

## 安装器说明

- Windows 安装器仍为交互式安装流程。
- 安装前会展示合规与免责协议。
- 用户可以自行选择安装目录。
- 协议文件继续使用 RTF 格式，避免 NSIS 中文乱码。

## 安全说明

当前如果尚未配置商业代码签名证书，浏览器、Windows 或 macOS 仍可能提示安全确认。请仅从官方 GitHub Release 页面下载安装包，并根据系统提示自行确认来源。

## 合规提醒

CloakBrowserDesktop 仅可用于合法、合规的学习、研究、测试或经授权场景。不得用于未授权访问、账号滥用、欺诈、攻击、规避平台规则或其他违法违规用途。
