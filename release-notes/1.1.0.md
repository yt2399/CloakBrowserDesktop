# CloakBrowser Desktop 1.1.0

## 更新概览

这是 CloakBrowserDesktop 的首个 GitHub Release 初始版本，用于提供 Windows 桌面安装包、GitHub Releases 分发和应用内更新检测基础能力。

## 新增功能

- 新增 Windows 桌面安装包发布流程，安装包由 GitHub Actions 云端构建并上传到 GitHub Release。
- 新增基于 GitHub Releases 的自动更新检测能力，正式打包环境会定期检查可用版本。
- 新增安装包构建来源证明，Release 产物会通过 GitHub Artifact Attestation 记录构建来源。

## 应用能力

- 支持本地浏览器环境管理。
- 支持内核版本列表、下载、安装状态查看和本地目录打开。
- 支持配置工作目录、浏览器环境、代理信息和启动参数。

## 安装说明

- 下载 Release 页面中的 Windows 安装包后运行安装。
- 当前版本安装包尚未使用商业代码签名证书，Windows 或浏览器可能提示“通常不会下载”或需要手动确认信任。
- 请仅从官方 GitHub Release 页面下载安装包，并核对 Release 版本号。

## 合规提醒

CloakBrowserDesktop 仅可用于合法、合规的学习、研究、测试或经授权场景。不得用于未授权访问、账号滥用、欺诈、攻击、规避平台规则或其他违法违规用途。
